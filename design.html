Hi! I'm Jimmy Jia

a Frontend Web Developer

If you're looking to get hire me, you're in the right place!

Design

All design is original, or supplied by the client. I can work with PSDs, and other types of mockups to create the website you want. I can also work with you to design a layout for you, including all the content you need.

Coding

All websites I create are written from scratch, no use of any frameworks like Bootstrap or Foundation. This way, we minimize the file size, avoiding any unneeded bloat that is included when you work with a framework

Pricing

My rates are very flexible, depending on the length of project, complexity, and whether you need maintenance. If you would like an estimate on your website or project, feel free to shoot me an email at jimmy.j.jia@gmail.com. I can work with almost any budget, though the lower it is, the less features the resulting website will have. I have options for both hourly rates and lump sums.

Features

All websites will be responsive (content resizes appropriately depending on the viewer's screen size) and optimized for fast loading. Code will be written entirely by me, without the use of frameworks.

Steps to Launch a Website

The basic requirements for a web page is a domain and hosting. (I can help you register for both) For creating the website itself, assets like images you'd like to use, possible layouts, and different features you'd like to implement are provided for me. Then I'll work with you to create a plan for what the site will look like, and upon approval, the initial versions of the design will be created in HTML/CSS. When the design is approved, more advanced features are implemented and everything is tweaked until you are satisfied.
